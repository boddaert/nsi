# Exercices 

## Exercice 1

Donner le résultat des expressions suivantes en précisant le type du résultat ou lorsqu'une erreur doit survenir.

- $5 ** 2 - 3$

- $5 / 2$

- $4 / 2$

- $4 / 0$

- $4 // 2$

- $4 // 0$

- $4 % 2$

- $( 3 + ) 5$

- $5 % 2$

- $7 + 7 % 2$

- $3 * 3 ** 2$

- $'4' // 2$

## Exercice 2

A l'aide de la console, déterminer l'opérateur le plus prioritaire entre :

- `*` et `**`

- `+` et `//`

- `%` et `*`
